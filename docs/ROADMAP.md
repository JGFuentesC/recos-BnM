# ROADMAP - Recos-BnM

Este documento detalla las fases de desarrollo del proyecto "Tinder de Contenidos" basándose en el PRD §13.

## Fase 1 (MVP)
* Épicas 1–5 con algoritmo de popularidad + contenido.
* Registro simplificado (Email/Google).
* Calibración inicial por swipe (cold start).
* Conmutador de tipo (Películas/Libros).
* Feed interactivo de tarjetas.
* Vista expandida con detalle y proveedores de streaming (TMDB Watch Providers).
* Colecciones, guardado y listas básicas con notas personales.

## Fase 2
* Señal de afinidad histórica: Mejora del algoritmo basado en el historial de swipes.
* Listas compartibles entre usuarios.
* Notificaciones push o en app.

## Fase 3
* Recomendación avanzada utilizando Machine Learning (Collaborative filtering).
* Verificación de disponibilidad de streaming geolocalizada en tiempo real.
* Estrategias de monetización.
